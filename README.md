# 🛍️ Olx Achadinhos

Uma loja digital focada na compra e revenda de eletrônicos usados, com qualidade garantida e excelente custo-benefício.

---

## 🚀 Sobre o projeto

O **Olx Achadinhos** foi desenvolvido com o objetivo de simular uma experiência real de marketplace, onde usuários podem visualizar produtos disponíveis, analisar detalhes e acessar o anúncio completo.

A proposta é oferecer uma interface moderna, simples e eficiente, priorizando:

- 💸 Preços acessíveis  
- 🔍 Transparência nas informações  
- ⚙️ Produtos testados e revisados  
- 📱 Experiência responsiva  

---

## 🧠 Funcionalidades

- 📦 Listagem dinâmica de produtos  
- 🔎 Filtro por categorias  
- 📱 Layout responsivo (mobile + desktop)  
- 🆕 Destaque para produtos recentes  
- 📄 Modal com detalhes do produto  
- 🔗 Redirecionamento para anúncio externo (OLX)  
- 🎯 Navegação com React Router  

---

## 🛠️ Tecnologias utilizadas

- ⚛️ React.js  
- 🎨 CSS3  
- 🔀 React Router  
- 📦 JavaScript (ES6+)
